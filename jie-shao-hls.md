# 介绍HLS

HTTP直播流协议提供一种可靠的、低成本的方式在互联网上传输连续的、长期协议的视频。为了在最佳质量下保持视频的不间断播放，它允许接收方根据当前网络条件适应媒体的比特率。它支持插页式内容边界\(It supports interstitial content boundaries\)。它为媒体加密提供的灵活的框架。它能够有效率的提供媒体内容的多种描述，比如，不同语言的音频翻译。为了面向大众传输，它兼容了应用广泛的HTTP缓存架构。

自从2009年首次发布互联网草案以来，HTTP直播流协议已由众多的内容生产商，工具供应商，发行商和设备制造商实施和部署。在随后八年里，通过与各种媒体实现者的广泛审查和讨论，协议得以完善。

本文档的目的是通过描述流媒体传输协议来促进HTTP直播流的互通性。客户端使用此协议可以从服务器接收连续的流媒体并同时进行显示。

这个文档描述协议的第七个版本。
